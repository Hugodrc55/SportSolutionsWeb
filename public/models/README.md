# Modèle 3D du ballon

Placez ici un **fichier .glb** de ballon de basket nommé exactement :

```
basketball.glb
```

## Où télécharger un modèle gratuit ?

- **Sketchfab** : [sketchfab.com](https://sketchfab.com) → recherchez "basketball" → filtre "Downloadable" → téléchargez en **GLB**
- **Poly Pizza** : [poly.pizza](https://poly.pizza) → modèles sport en GLTF/GLB
- **CGTrader** (gratuit) : [cgtrader.com](https://www.cgtrader.com/free-3d-models/sports/equipment/free-basketball-ball)

Après téléchargement, renommez le fichier en `basketball.glb` et déposez-le dans ce dossier.

Si le fichier est absent, le site affichera un ballon 3D simple (sphère avec lignes).
